# ShoesHub - Complete E-Commerce Platform

A modern, full-featured e-commerce web application for an online shoe shopping website with both **Customer Panel** and **Admin Panel**.

## 🎨 Design & Features

### Style & Design
- **Clean, modern, minimal UI** with mobile-first responsive design
- **Color Palette**: Black/Dark primary, Orange accent, Neutral whites and grays
- **Typography**: Clean and readable with consistent spacing
- **Soft shadows**, rounded corners, and smooth transitions
- **Fully responsive** for mobile, tablet, and desktop
